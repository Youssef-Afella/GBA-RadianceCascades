# GBA-RadianceCascades
Implementation of Radiance Cascades on GBA (WIP)

Just a little project cause I was bored, and I always wanted to experiment with GBA game dev</br>
I wanted to see how far can I optimize the implementation to run on the GBA hardware.</br>
Still not fast enough, it renders one frame each 3 seconds on the real hardware.</br>
Maybe I will improve it, maybe not.
